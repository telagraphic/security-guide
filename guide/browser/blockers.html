Browser Fingerprint

Your web browser sends information about your computer platform, timezone, screen size, and many other details that advertisers use to track you. Plug these data points into a database and perform some analysis and your identity can be tracked and possibly forecasted. Health Websites track your queries when you search on your medical symptoms.

The tools listed below should not all be used at the same time, usually one or two is sufficient. Simply downloading all of them in the same browser is not the goal.

Privacy Badger blocks certain ads while NoScript and uBblock Origin require configuration for all the types of scripts and ads to be blocked. The list below goes from simple and automatic to more complex and manual.

Unfortunately, even ad blockers and browser extensions designed to protect the user are not 100% secure. There are so many ways to track a user according to a new report . But something is better than nothing.

Privacy Badger

Free

Desktop

The Electronic Frontier Foundation (EFF) is a leader in tech security and privacy. They built Privacy Badget to track your web searching and block ads that track you. Specfically, it blocks trackers that do not respect the Do Not Track http header.

Check out the homepage for detailed information . Follow on Twitter for updates.

  • Blocks Tracking Ads

Enfore HTTPS Everywhere

Free

Desktop

HTTPS is simply regular old HTTP but encrypted. It is a new standard being widely adopted but not all sites use it . Typically, the browser toolbar will show an icon showing if it is HTTPS or not.

A collboration from the EFF and the TOR project, HTTPS Everywhere finds and fixes unencrypted links and content to use HTTPS.

  • Updated Frequently

uBlock Origin

Free

Desktop

uBlock Origin is a trusted tool used by privacy minded inviduals. It's also more than just an 'Ad Blocker', it can block scripts, zap annoying elements on a page, and whitelist certain webpages.

It is a more advanced tool that requires some learning and configuration, but works right of the box as an adblocker. It has a visual pop out box that displays all the scripts and trackers for each webpage and site. It's refreshing to see how the code behind the scenes is working, it's also scary how much stuff is tracking you!

Random User Agent

Free

Desktop

Random User Agent sends a new User-Agent string at a recurring period. Spoofing your user agent makes your browser harder to identify as you.

Your browser sends the browser name and version, the devices OS system and version, and other information used for identification.

Funny enough, just changing the User Agent string may make you more identifiable. . This tool is probably not ready for real personal use.

Decentral Eyes

Free

Desktop

Many of the scripts served on websites are hosted on CDN's or Content Delivery Networks that aim to increase speed and performance. When the browser sends a request to the CDN asking for the file, it also sends an HTTP Referer header, which reveals the page you are on.

Instead of loading third party javascript files, Decentral Eyes will provide the script from its files thus preventing this data leak.

  • Works on Firefox, Opera and Chrome
  • Works Out of the Box!
  • Protects Against CDN Tracking
  • Compliments Other Ad Blockers
