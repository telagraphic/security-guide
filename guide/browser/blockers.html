Browser Fingerprint

Your web browser sends information about your computer platform, timezone, screen size, and many other details that advertisers use to track you. Plug these data points into a database and perform some analysis and your identity can be tracked and possibly forecasted.

The tools listed below should not all be used at the same time, usually one or two is sufficient. Simply downloading all of them in the same browser is not the goal.

Privacy Badger blocks certain ads while NoScript and uBblock Origin require configuration for all the types of scripts and ads to be blocked. The list below goes from simple and automatic to more complex and manual.

Don't Trust

Ghostery, a popular ad blocker has been selling data to advertisers. User beware! Here is a study on ad blocker software. Ad Blocker Plus was caught doing the same thing years ago.

Website owners and companies use services that can record your screen for analytics and to see how customers use their website. But that also means they could capture you entering passwords or confidential information! Check out this list of sites that have session replay scripts. There really isn't anything you can do about this unfortunately!

Privacy Badger

Free

Desktop

The Electronic Frontier Foundation (EFF) is a leader in tech security and privacy. They built Privacy Badget to track your web searching and block ads that track you. Specfically, it blocks trackers that do not respect the Do Not Track http header.

Check out the homepage for detailed information. Follow on Twitter for updates.

  • Works In Firefox, Chrome, Opera
  • Blocks Tracking Ads
  • Updated Frequently

Enfore HTTPS Everywhere

Free

Desktop

HTTPS is simply regular old HTTP but encrypted. It is a new standard being widely adopted but not all sites use it. Typically, the browser toolbar will show an icon showing if it is HTTPS or not.

A collboration from the EFF and the TOR project, HTTPS Everywhere finds and fixes unencrypted links and content to use HTTPS.

  • Works In Firefox, Chrome, Opera
  • Updated Frequently

uBlock Origin

Free

Desktop

uBlock Origin is a trusted tool used by privacy minded inviduals. It's also more than just an 'Ad Blocker', it can block scripts, zap annoying elements on a page, and whitelist certain webpages.

It is a more advanced tool that requires some learning and configuration, but works right of the box as an adblocker. It has a visual pop out box that displays all the scripts and trackers for each webpage and site. It's refreshing to see how the code behind the scenes is working, it's also scary how much stuff is tracking you!

uBlock has been field tested and improved to combat ad blocking. There site contains much information about ad blocking and how it all works. Click on the links below to school up on this awesome tool!

Disconnect

Free and Paid Options

Desktop & Mobile

A New York Times Privacy tool pick for the past few years, Disconnect offers a free basic ad tracker and paid options that include a VPN and multiple device integration. It is very straightforward to use for the non-technical and shows how much bandwidth you save when ads are blocked.

  • Works in Firefox, Chrome, Safari, Opera, Android and iOS
  • Ad Blocker
  • Prevents Clickjacking
  • Advanced Features

Track Off

Paid

Desktop & Mobile

Track Off aims to be a one stop shop for all browser security needs. There are lots of cool features that are combined from the above the tools.

It does cost money but is not expensive. It is perfect for the set it and forget non-technical user, it looks to be a solid product. It offers a user-friendly dashboard UI unlike most browser tools.

  • Works in Most Browsers
  • Reduces Browser Fingerprint
  • VPN
  • Dashboard UI That Displays Threats
