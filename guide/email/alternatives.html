Proton Mail

Free & Paid

Desktop & Mobile

Proton Mail is built by scientists and engineers who met at CERN. A user friendly interface backed by end-to-end encryption with an anonymous sign up process makes this the go to email for security and privacy.

It's based in Switzerland which maintains strict privacy laws. User data is impossible to access, so don't lose your password.

  • Desktop and Mobile
  • Open Source Encryption
  • Zero Access to User Data
