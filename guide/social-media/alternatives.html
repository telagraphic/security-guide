Alternatives

It's not easy to overcome the power of Facebook and Twitter. How about a social network that isn't overcrowded and filled with the lowest common denominator of human behavior. Check out some Facebook and Twitter alternatives.

Ello

Desktop & Mobile

Ello is network for creatives, artists and designers. It is niche network, but one that doesn't sell user data, has no advertisements, and doesn't collect user information!

  • Doesn't Sell User Data to Advertisers
  • No Advertisements
  • Doesn't Enforce Real Name Policy
Goodbye Instagram, Hello Ello
Guide to Ello

Mastodon

Desktop & Mobile

Mastodon is microblogging community similar to Twitter. It is run on an decentralized network of independently run servers.

  • Decentralized Network
  • Community Based, Not Top Down
  • Runs on OS Code

Diaspora

Desktop & Mobile

Diaspora gives the user ownership of their data! Of course, this requires a little more effort to set up an instance and getting connected.

Diaspora got initial funding with a Kickstarter campaign.

  • Decentralized Network
  • You Own Your Data
  • TOR Integration
